# koydo-cet-6

CET-6 exam preparation — powered by Koydo.

## Quick start

```bash
npm install
npm run dev
```

## Stack

- **Framework:** Next.js 16.2.0 + React 19
- **Styling:** Tailwind CSS 4
- **Backend:** Supabase (shared)
- **Deploy:** Vercel

## Exam Metadata

| Field | Value |
|-------|-------|
| Exam ID | EXAM072 |
| Name | CET-6 |
| Country | China |
| Category | Language proficiency |
| Tier | Tier 3 |
| Wave | Wave 5 |
